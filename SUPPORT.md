# Support

LocalCloud is provided for **evaluation and testing purposes only** and is provided **"AS IS"** with **no warranties or guarantees**. Support is **best-effort** and response times are not guaranteed.

## How to get help

1. Check the documentation: https://docs.localcloud.dev
2. Search existing issues in this repository.
3. Open a new issue with the requested diagnostics.

## What to include in bug reports

- `localcloud --version`
- Install method (brew/apt/winget/curl|sh/etc.)
- OS + architecture
- `localcloud doctor` output (redact secrets)
- Steps to reproduce (minimal repro), expected vs actual behavior
- Relevant logs (redact secrets)
