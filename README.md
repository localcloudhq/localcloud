## Filing a Bug

Please include:

- `localcloud --version`
- Install method (brew/apt/winget/curl|sh/etc.)
- OS + architecture
- `localcloud doctor` output (redact secrets)
- Minimal repro steps + expected vs actual behavior
- Relevant logs (redact secrets)

## Security Reports

Do **not** open public GitHub issues for security vulnerabilities. Email: support@eyequs.com (subject: `SECURITY: <summary>`).
